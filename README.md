## Description

Construção de uma API de filmes em TYPESCRIPT e NESTJS

## Clonando o repositório

```bash
$ git clone https://github.com/diegovenas/Blue-M4.git
```

## Instalação

```bash
$ npm install
```

## Iniciando app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
